# BaDumTss.

This is yet another MIDI-drum implementation. 

It uses a highly similar CMake configuration to the [Raspberry Pi Examples][examples], so should be compilable in a very similar way: ["Getting started with Raspberry Pi Pico" document][picostart].

[examples]: https://github.com/raspberrypi/pico-examples/
